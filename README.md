# elevenbeans.me

Personal profile page. Built with vanilla HTML, CSS, JS — zero dependencies.

## Features

- Work timeline & project showcase
- Dark/light theme with system preference detection
- i18n (EN / 中文)
- Hidden retro terminal (双击 / Ctrl+`)
- Hidden code agent easter egg (Ctrl+Shift+`)

## Deploy

Push to `main` — GitHub Pages auto-deploys to elevenbeans.me.
